Best Apps of Linux

Thu 04 February 2016

This is my curated list of all the best apps that I found on linux while my journey with it. The list may extend or shorten as per my later experiences.

Programming

  • Atom

    The cool space age text editor. atom.io

  • Jupyter

    The scientific Pythoning web-browser interface. Project Jupyter

  • Scipy

    The python Scientific library. SciPy.org

  • Koala

    For automatic compiling of style sheets. Koala

Multimedia

  • Lollypop

    Crazy good music player. Lollypop

  • Tomahawk

    Another nice music player with support for online services. Tomahawk

  • Pulse audio equalizer

    System wide audio equalizer. Pulse Audio - webupd8

Graphics

  • GIMP

    The best photo editor there is. Gimp

  • Inkscape

    The amazing vector graphic app. Inkscape

  • MyPaint

    If you liked painting on MSPaint and want something better. MyPaint

  • Krita

    MyPaint isn't enough, try this and cry your eyes out. Krita

Internet and Communication

  • Firefox

    Since nothing's better. Firefox

  • Chromium

    Just to Draw tracks for Free Rider. Chromium

  • Telegram

    This thing is the standard messenger. Telegram

  • Corebird

    The twitter client for twitter addicts. Corebird

Utilities

This is great for power users

  • Gnome-Pie

    A little app that launches great apps. Gnome Pie

  • Albert

    The search launch app. Choose synapse if you need more power. Albert

  • Terminator

    The exterminator of the terminal emulators. Terminator

  • FromScratch

    The great scratch pad, one file, super cool auto save, Fira Code And nothing else. FromScratch

  • Variety

    Never change walls yourself, It will do that for you. Variety

  • Shutter

    Because Your screen shot matters. Shutter

  • Qalculate

    Math the problem away. Qalculate

Terminal

  • oh-my-git

    This is a simple bash addon which gives a powerline interface for git repos. oh-my-git

  • fortune | cowsay | lolcat

    This triad gives beautiful message or fortune, great as a start script.
